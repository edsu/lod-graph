Linked Open Data Cloud

This is a Protovis version of the (much more useful) Linked Open Data Cloud. It is based on the same underlying data, by using the Comprehensive Knowledge Archive Network (CKAN) API to pull all the packages in the the lodcloud group. If your screen goes crazy cross your fingers and try to reload the page a couple times. The colors reflect the CKAN rating: (0, 1, 2, 3, 4, 5). The size of a node is based on the connectedness of the dataset, not on the number of triples the dataset contains. You should be able to hover over the node to get its title. More about how this graph was generated can be found on Github. Comments/questions welcome at ehs@pobox.com.
